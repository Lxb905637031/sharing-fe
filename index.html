# 🛋 配套工具

引入了多个业界流行的 Linter 作为规范文档的配套工具，并根据规范内容定制了对应的规则包，包括：

规范 Lint工具 npm包
JavaScript 编码规范
TypeScript 编码规范
Node 编码规范		 ESLint (opens new window) eslint-config-sharing (opens new window)
CSS 编码规范 stylelint (opens new window) stylelint-config-sharing (opens new window)
GIT 规范 commitlint (opens new window) commitlint-config-sharing (opens new window)
文档 规范 markdownlint (opens new window) markdownlint-config-sharing (opens new window)
ESlint 插件 ESLint Plugin (opens new window) eslint-plugin-sharing (opens new window)

sharing-fe-lint (opens new window) 收敛屏蔽上述依赖和配置细节，提供简单的 CLI 和 Node.js API,让项目能够一键接入、一键扫描、一键修复、一键升级，并为项目配置 git commit 卡口，降低项目接入规范的成本
